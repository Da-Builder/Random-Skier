- I am using Node.js. Node.js is a JavaScript runtime outside the browser that allows development tools to run, and its bundled with lots of nice development tools like npm (package manager) and npx (runs exectubles from installed packages). - Note that the JavaScript code for the actual application runs in browser runtimes.

- The `const` specifier in JaveScript declares a constant binding. That is, the content of the object can change, but you cannot assign it to another object later.

- Document Object Model (DOM) is the in‑memory tree representation of your HTML page that JavaScript can read and modify - the browser turns your HTML into a live data structure (a tree) that JavaScript can manipulate.

- In JavaScript, a value is an umbrella term for objects and primatives (things like integers are primatives, not objects, because there is no such thing as an int class in js)

- In JavaScript, a function is a "callable object". By not putting parentheses, it's a reference to the function object and by putting parentheses it's the value the function returns after executing
